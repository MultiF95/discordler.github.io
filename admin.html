General Stuff

How to start a game server?
Go to OGP and user your login credentials or use the user you've got and it's password.
Then go to "Game Monitor" and press the game server you want to start. Scroll down until you see "Server starten" / "Start Server". It will now start.
How to start rust server?
Open putty and start a connection to discordler.ddns.net with the username "rust" and it's password. Then type in the following:

cd /home/rust/rust1
./rustserver start
